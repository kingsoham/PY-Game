# 🏎️💨 Gravity Rider

**Gravity Rider** is a fast-paced, highly polished 2D antigravity racing game built entirely natively using Python and Pygame primitives! 🕹️

Experience deep parallax scrolling 🏔️, violent crashes with heavy screen shake 💥, and smooth particle effects ✨—all completely generated via code without the need for external image assets!

## ⭐ Features
- **🔥 Dynamic Particle Engine**: Advanced custom particle systems for realistic exhaust trails, interactive coin collection sparkles, and high-impact crash bursts.
- **🌌 Parallax Backgrounds**: Infinite 6-layer mountain parallax generation that perfectly simulates depth relative to your camera speed!
- **⛰️ Rich Vector Terrain**: Multilayered procedural polygon terrain featuring deep shadowed bottoms and brightly highlighted edges.
- **💢 Impact Feedback**: Action-camera style heavy screen shake triggered upon severe impacts and game over events.
- **🖥️ Fluid UI**: Clean, drop-shadowed text interfaces and dynamically changing fuel gauges that alert you when gas is low!

## ⚙️ Requirements
- 🐍 Python 3.10+
- 🎮 Pygame-CE (`pip install pygame-ce`)

## 🏁 How to Play
1. Clone the repository to your local machine. 💻
2. Run `python gravity_rider.py` from your terminal or favorite IDE. ▶️
3. Grab yellow coins 🟡 to increase your score and snatch green fuel cans 🔋 BEFORE your gas runs out!

**⌨️ Controls:**
*   `→` / `D`  -  Accelerate ⏩
*   `←` / `A`  -  Brake & Reverse ⏪
*   `R`  -  Restart Match 🔄
*   `ESC`  -  Quit Game 🚪

## 🛠️ Installation
```bash
git clone https://github.com/kingsoham/PY-Game.git
cd PY-Game
pip install pygame-ce
python gravity_rider.py
```
